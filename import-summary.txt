Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

In BetterPicker:
* AndroidManifest.xml => betterPicker/src/main/AndroidManifest.xml
* assets/ => betterPicker/src/main/assets/
* libs/nineoldandroids-2.4.0.jar => betterPicker/libs/nineoldandroids-2.4.0.jar
* lint.xml => betterPicker/lint.xml
* res/ => betterPicker/src/main/res/
* src/ => betterPicker/src/main/java/
In Bolts:
* AndroidManifest.xml => bolts/src/main/AndroidManifest.xml
* lint.xml => bolts/lint.xml
* res/ => bolts/src/main/res
* src/ => bolts/src/main/java/
In Calligraphy:
